# 穷游网

-- 穷游为旅行者提供全球热门出境游目的地和新奇小众景点，针对特色美食，当地特色的活动和购物攻略等旅行资讯，该 APP 既提供来自千万用户的真实点评和图片，也提供来自穷游锦囊、旅行达人的推荐榜单。出境旅行者可以随时查看附近的吃喝玩乐目的地，也能轻松购买全球特价机票、酒店、景点门票、一日游、特色活动、当地交通卡、电话卡等旅行产品。
